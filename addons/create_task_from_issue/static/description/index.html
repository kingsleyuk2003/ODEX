Create tasks from issues

Automatically create a task based off one of your issues

This module allows to convert an issue to a task by a simple button click.

The new task will be pre filled with all the necessary information (Assigned to, Project, Description and Issue title).

This module also adds a tab with the list of issues linked to the task.
